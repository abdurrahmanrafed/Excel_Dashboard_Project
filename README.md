# Excel_Dashboard_Project
Project_Name: Amazon Sales Dashboard |
Used_Tools: Excel Pivot tables, Power Query

📊 Excited to share this interactive Excel dashboard, built using Pivot Tables, that provides an in-depth analysis of Amazon's sales performance across multiple dimensions.
The dashboard breaks down sales data by region, state, city, and individual sales representatives, offering valuable insights into:
- Regional trends and opportunities
- Top-performing states and cities
- Individual sales rep contributions and metrics
